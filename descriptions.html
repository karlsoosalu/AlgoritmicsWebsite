QUICK SORT

✓ Pick one element as a pivot (middle one in the current implementation).

✓ Partion the list. Iterate through elements and swap them in the way smaller elements end on the left side of the pivot and larger elements on the right.

✓ When it is partioned, quicksort left and right part of the pivot separately.

MERGE SORT

✓ Divide the list in half until list is desired size (1 in current implementation).

✓ Mergesort both halves separately.

✓ Merge both halves, so they end up in sorted order

HEAP SORT

✓ Build a Minheap, where root is always the smallest element.

✓ Insert elements to the heap.

✓ Take the elements out of the heap one by one.

BUBBLE SORT

✓ Compare and swap two neighboring elements until they are in the intended order.

INSERTION SORT

✓ Iterate through all the elements

✓ Compare all the elements to previous ones and place it to the correct position

SELECTION SORT

✓ Find the smallest element in each iteration and replace it with the index of iteration.

RADIX SORT

✓ For each digit from least significant to the most significant sort input array according to that digit using counting sort.

BUCKET SORT

✓ Create buckets according to min and max value.

✓ Put all objects to correct bucket.

✓ Sort buckets separately.

JAVASCRIPT BUILT-IN SORT

✓ Takes a compareFunction(a,b) as argument, by default sorts integers as strings.

✓ Implementation differ in various browsers. Usually quicksort alternative is used.
